ARChemy Mobile Web Frontend

Cara pakai:
1. Buka index.html di browser.
2. Pilih role Siswa atau Guru di halaman login.
3. Untuk siswa, kode kelas dummy: KIM-137.
4. Data dummy tersimpan di localStorage browser.

Fitur:
- Login dan daftar role siswa/guru
- Siswa: gabung kelas, dashboard, modul, ringkasan AI, kuis adaptif 4-tier, rekomendasi AI, WebAR placeholder dengan kontrol suhu/tekanan/konsentrasi/volume, mode inklusif warna, profil/edit profil/logout
- Guru: pilih kelas slider, buat kelas dan kode, dashboard ringkasan, kelola modul CRUD, kelola kuis, tambah/edit/hapus soal, analisis pembelajaran, profil/edit profil/logout

Catatan:
Frontend ini belum memakai backend/database. Semua interaksi masih dummy di sisi browser.
